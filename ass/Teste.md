`como usar as branchs do Github e como dar commit/push`
___
# Significados:
	- Commit = Prepara os arquivos para enviar para o repositório remoto;
	- Clone = Copia o repositório para seu pc;
	- Pull = Pega as atualizações que já sofreram push e atualiza o seu
	repositório da sua máquina com eles;
	- Push = Envia seus arquivos preparados (commits) para o repositório 
	remoto;
___

# Dicas:
	- Sempre que fizer um commit, você precisar dar um nome/título a ele
	(Resumido) e também se você quiser ser bonzinho, coloca as alterações
	que fez na descrição do commit por favor;
	- Sempre conferir antes de mexer no projeto, se tem alguma alteração,
	dá um pull somente por que eu vou dar merge sempre com suas branches
